# v1.0.1 — 公网共享增强、Web 分享体验改进、智能缩略中图

PhotoPeek 首个正式功能版本！基于 Electron + React 的本地照片管理与共享应用，支持局域网和公网实时分享照片。

---

## ✨ 新功能

### 📸 照片管理

- **事件管理** — 以事件（文件夹）为单位组织照片，支持创建、重命名、删除
- **照片导入** — 从文件夹批量导入照片，自动读取 EXIF 元数据（拍摄日期、相机型号等）
- **标签系统** — 为照片添加彩色标签，支持按标签筛选
- **照片评级** — 在大图模式下按 `1`-`5` 快速评级
- **回收站** — 软删除照片，可恢复或永久清空
- **缓存管理** — 一键清除所有缓存（数据库、缩略图、导入照片）

### 🖼️ 浏览体验

- **虚拟网格** — react-virtuoso 驱动，流畅浏览大量照片
- **Loupe 大图预览** — 支持键盘快捷键（`←` / `→` / `j` / `k` 切换，`Esc` 退出，`Space` 标记标签）
- **缩略图缓存** — 自动生成并缓存缩略图，加速浏览
- **中文 / Unicode 路径兼容** — 自动处理含非 ASCII 字符的文件路径

### 🌐 Web 共享（全新）

- **局域网共享** — 一键启动 HTTP + WebSocket 共享，同网络设备浏览器即可查看
- **Bonjour / mDNS 自动发现** — 局域网内自动广播共享服务
- **公网共享（Cloudflare Tunnel）** — 生成 `trycloudflare.com` HTTPS 公网地址，无需端口转发
- **实时协作** — 访客昵称系统，双击添加标签，操作实时同步到所有设备
- **三档画质** — Web 端大图默认加载**中图**（1200px，JPEG 65 品质），用户可点击「加载原图」查看全分辨率并显示加载进度
- **打包下载** — Web 端一键打包下载所有照片为 ZIP
- **二维码访问** — 手机扫码快速进入共享页面
- **在线访客管理** — 实时显示在线用户列表

### 🔧 系统优化

- **sql.js WASM 数据库** — 替代 better-sqlite3，避免原生模块编译问题
- **`photo://` 自定义协议** — 安全加载本地图片
- **macOS 风格窗口** — `titleBarStyle: 'hiddenInset'` 无边框窗口
- **快捷键系统** — 类 Vim 键位支持（`j` / `k` 导航）
