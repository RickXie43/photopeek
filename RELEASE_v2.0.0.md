# v2.0.0 — 多版本管理 + RAW 预览 + Web 对比模式

PhotoPeek v2.0.0 重磅更新！引入完整的照片多版本管理系统，支持 RAW+JPEG 自动合并、版本对比、Web 端上传编辑版本。

---

## ✨ 新功能

### 📸 多版本管理

- **版本分组** — 导入时自动根据 EXIF 拍摄时间 + 文件名将 RAW 与相机 JPEG 合并为同一照片的多个版本
- **版本面板** — Loupe 大图模式右侧版本面板，清晰展示所有版本（RAW、JPEG、编辑版等）
- **三种对比模式**：
  - **并排对比**（Side-by-side）— 左右并排显示两个版本
  - **滑动对比**（Slide）— 拖拽分割线滑动比较
  - **切换对比**（Toggle）— 鼠标悬停快速切换前后版本
- **版本上传** — Web 端可直接上传编辑后的新版本，自动以 `昵称_序号` 命名
- **智能默认** — 进入大图时自动选中非 RAW 版本（优先显示 JPEG/编辑版）

### 🌐 Web 共享增强

- **版本路由** — 所有图片路由（缩略图/中图/原图）支持 `?version=` 参数
- **版本上传端点** — `POST /upload/{photoId}` 支持从浏览器上传版本
- **Web 版本面板** — 勾选版本进行对比，支持并排/滑动/切换三种模式
- **版本徽章** — 网格视图中照片角标显示版本类型（RAW/JPEG/编辑者）
- **实时同步** — 上传版本后自动广播到所有连接设备
- **打包下载** — ZIP 下载包含所有版本

### 🖼️ RAW 格式预览

- **RAW 嵌入 JPEG 提取** — 通过 exifr 从 RAW 文件中提取内嵌 JPEG 预览图
- **photo:// 协议增强** — 直接渲染 RAW 文件的预览图像
- **维度检测** — 自动读取 RAW 文件的原始分辨率
- 支持 CR2、CR3、NEF、ARW、RW2、ORF、RAF、DNG 等主流 RAW 格式

### 🔧 桌面端增强

- **元数据恢复** — 上传到桌面端的版本自动从文件读取 EXIF 并恢复元数据
- **维度自动检测** — 上传版本使用 sharp 自动检测图片尺寸
- **中图质量优化** — 1200px / JPEG 80 品质，画质与体积均衡
- **文件名排序** — 工具栏新增排序切换（按拍摄时间 / 按文件名）
- **输入焦点保护** — 全局快捷键自动跳过输入控件，避免对话框输入被拦截

### 🛠️ 系统优化

- **检查更新** — 设置 → 关于新增检查更新按钮，从 GitHub Releases 获取最新版本
- **缓存管理细分** — 设置 → 缓存管理提供「清除所有数据和设置」选项
- **Unicode 路径兼容** — 中图/大图生成（resizeWithSharp）增加 Unicode 路径处理，避免中文路径报错
- **模块加载优化** — 修复 Vite 打包后动态 require 失败导致的静默降级问题

## 🐛 Bug 修复

- 修复 Web 端上传后黑屏问题（中图缓存未预生成）
- 修复 RAW 文件在 photo:// 协议中无法显示的问题
- 修复跨会话重复导入时唯一约束冲突

- 修复对话框中文本框无法输入文字（移除 `document.addEventListener` + 捕获阶段监听器干扰）
- 修复 Web 上传版本未设置 `uploadedBy` 修改用户
- 修复 Web 上传/删除版本后桌面端版本标签未实时刷新
- 修复上传版本分辨率显示为 0×0
- 修复对话框输入框被全局快捷键拦截的问题
- 修复中图质量过高导致加载缓慢
